# Sanchez Restore and More

Professional web presence for an automotive business focused on paint and body work, detailing, and maintenance services.

## Overview

This project is a React + Vite application that presents:

- Branded hero and value proposition
- Core service categories
- Business process section
- Trust and credibility highlights
- Contact call-to-action

The design direction is intentionally retro and professional, with custom typography, layered backgrounds, and bold section framing.

## Tech Stack

- React 19
- Vite 8
- ESLint 10

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## Available Scripts

- `npm run dev`: Starts Vite dev server
- `npm run build`: Creates production build
- `npm run preview`: Serves production build locally
- `npm run lint`: Runs ESLint checks
